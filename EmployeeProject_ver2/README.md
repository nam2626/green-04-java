# EmployeeProject
---
간단한 사원 관리 프로그램
관리되는 사원 정보는 **사번**,**이름**,**직급**,**급여**,**입사일**

## 프로젝트 구조
```
    controller
      ┝ Controller
      ┝ HandlerMapping
      | ...
    main
      ┝ EmployeeMain
    service
      ┝ EmployeeService 
    exception
      ┝ EmplpoyeeException
    vo
      └ EmployeeVO   
```
