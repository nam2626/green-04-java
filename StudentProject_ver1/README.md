# 학생관리 프로젝트 v1

## 클래스
- StudentVO : 학생정보 저장하는 클래스
- StudentService : 학생정보를 관리하는 클래스
- StudentMain : 사용자가 입력하여 학생정보를 관리하는 인터페이스

## StudentService에서 하는 일
- 학생 정보를 배열로 관리
- 학생정보 추가
- 학생정보 삭제
- 학생정보 수정
- 학생정보 조회

## StudentVO
- 학번, 이름, 평점, 학과명
- 필드 초기화하는 생성자
- 학생정보 출력하는 메서드
- 학생정보 수정하는 메서드

## StudentMain
- 메뉴 출력
- 사용자로부터 숫자 입력 받아서 StudentService의 메서드 실행
